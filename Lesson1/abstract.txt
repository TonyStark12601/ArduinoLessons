Сегодня мы начнем осваивать программну ючасть, в частности типы данных.
Прошивка для Arduino называется скетч.Скетчи имеют расширение - (.ino) и хранятся в папке с именем как сам скетч.
Arduino программируется на языке - Arduino viring который является упрощенной версией C++ и наследует от него синтаксис.Когда вы нажимаете кнопку компиляции , программа компилируется и превращается в машинный код, который загружается в Arduino, этот процесс необратим, то есть достать готовую прошивку из Arduino уже не возможно. Структура прошивки такая :

1) boolean tmp; (занимает 1 байт , диапазон : 0 или 1 , логическая переменная может принимать значение true(1) или false(0))
можно инвертировать с помощью восклицательного знака true = !false
2) char tmp;(занимает 1 байт , диапазон : -128...127 , хранит номер символа из таблицы символов ASCII)
3) byte tmp;(занимает 1 байт , диапазон : 0...255 , используется для хранения небольших положительных чисел.)
4) int tmp;(занимает 2 байта , диапазон : -32768...32767 , )
5) unsigned int tmp; (занимает 2 байта , диапазон : 0...65535 , )
6) word tmp(то же самое что и unsigned int)
7) long tmp; (занимает 4 байта , диапазон : -2 147 483 648...2 147 483 647 , )
8) unsigned long tmp; (занимает 4 байта , диапазон : 0...4 294 967 295 , )
9) float tmp; (занимает 4 байта , диапазон : -3.4028235E+38...3.4028235E+38 , хранит числа с плавающей точкой, точность : 6-7 знаков )
переменной с типом float - обязательно присваивать нужно только число с точкой , даже если оно целое 
float number = 10.0;
а если вы захотите делить переменную типа float на другое, то опять же делить надо на число с точкой иначе результат будет целым числом.
float number  = 10.0;
float number1 = number / 2.0;
Если мыделим переменную целого типа и хотим получить дробное значение и записать его в тип float , то перед переменной надо поставить (float)
int number0 = 10;
float numnber2 = (float)number0 / 2.123;
это нужно для того , чтобы компилятор выделил из памяти 4 байта для проведения вычисления чисел с плавающей точкой , операции с числами типа float занимают намного больше времени , чем с остальными типами данных , так что использовать надо с умом , иногда выгоднее умножить число на 1000 - посчитать все в больших числах, а затем снова разделить на 1000.Также можно выводить числа с плавающей точкой с указанным количеством цифр после запятой
